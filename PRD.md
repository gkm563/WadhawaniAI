Problem Statement:

Goal:
Build an AI-powered multilingual employability assistant for underserved students in India.

Project Name:
PathPilot AI

Core Vision:
A low-bandwidth, AI-powered employability and career guidance platform designed for students from Tier-2/Tier-3 colleges, rural backgrounds, and underserved communities.

The platform should help students:

* identify skill gaps
* improve employability
* prepare for interviews
* receive personalized career roadmaps
* access learning guidance even in low-connectivity environments

The solution must align with these hackathon design principles:

* offline-ready
* low-connectivity support
* entry-level smartphone compatibility
* multilingual support
* scalable architecture
* human-centered AI
* practical implementation feasibility
* responsible AI usage
* low compute requirements

# CORE FEATURES (STRICTLY LIMIT TO THESE)

1. AI Skill Gap Analysis

* Student uploads resume OR manually enters skills
* AI compares against target job role
* Detects missing skills
* Suggests improvement areas

2. Personalized Career Roadmap Generator

* Generates 30-day / 60-day roadmap
* Suggests:

  * projects
  * certifications
  * DSA topics
  * interview preparation
  * aptitude practice
* Multilingual output support

3. AI Mock Interview

* Text-first interview mode
* Optional voice support
* AI asks technical + HR questions
* Evaluates:

  * clarity
  * confidence
  * technical quality
* Generates feedback report

4. Offline-First Learning Kit

* Downloadable learning packs
* Cached interview questions
* Lightweight UI
* PWA support
* Offline access for previous reports
